## Abstract

The Advanced BMI Calculator is a desktop-based healthcare application developed using Python. The application calculates the Body Mass Index (BMI) of users based on their weight and height. It further categorizes users into different health classifications such as Underweight, Normal Weight, Overweight, and Obese.

The system stores user information in a database, allowing historical record maintenance and retrieval. It also provides graphical trend analysis and statistical insights to help users monitor their health status over time.

---

# Chapter 1: Introduction

Body Mass Index (BMI) is a widely used measure to assess an individual's body weight relative to height. It helps determine whether a person is underweight, normal weight, overweight, or obese.

This project automates the BMI calculation process and provides advanced functionalities such as data storage, historical tracking, and graphical analysis.

# Chapter 4: System Design

## Input

* User Name
* Weight (kg)
* Height (m)

## Processing

BMI = Weight / Height²

BMI Classification:

* Underweight (<18.5)
* Normal Weight (18.5–24.9)
* Overweight (25–29.9)
* Obese (≥30)

## Output

* BMI Value
* Health Category
* Stored Database Record
* Statistical Analysis
* Trend Graph
